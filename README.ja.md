# API Key Manager

[简体中文](README.md) | [English](README.en.md) | [繁体中文](README.zh-TW.md) | [日本語](README.ja.md)

開発者がAPIキーを効率的にバッチインポート、グループ化、テスト、クリーンアップできるローカル実行のAPIキー管理ツール。

## 機能

- **バッチインポート** — 複数のキーを貼り付けるだけで自動重複排除、Base64エンコードされたキーの自動デコード対応
- **グループ管理** — プロバイダーや用途でキーをグループ化、カスタムグループ集計ビュー対応
- **ワンクリックテスト** — マルチスレッド並行でのキー可用性テスト、レイテンシとエラー詳細を表示
- **統計ダッシュボード** — 利用可能 / 失敗 / 合計キー数を一目で確認
- **クリーンアップ** — 失敗したキーをすばやく削除してキープールをクリーンに保つ
- **ライト / ダークテーマ** — システム設定に自動追従

## 技術スタック

| レイヤー | 技術 |
|---|---|
| バックエンド | Python / Flask |
| フロントエンド | 単一ファイル HTML（フレームワークなしのVanilla JS） |
| データベース | SQLite |
| APIプロトコル | OpenAI互換エンドポイント |

## クイックスタート

### 前提条件

- Python 3.9+

### インストールと実行

```bash
# 依存関係のインストール
pip install -r requirements.txt

# サーバー起動
python app.py
```

起動後、ブラウザで http://localhost:5000 を開きます。

Windowsユーザーは `start.bat` をダブルクリックしても起動できます。

## ワークフロー

1. **グループ作成** — APIプロバイダー（例：OpenAI、Claude）ごとにグループを作成し、プロトコルとモデルを設定
2. **キー追加** — グループにキーをバッチ貼り付け、追加時に自動テスト
3. **状態確認** — ダッシュボードで全体の状態を確認、またはグループ内の個別キー詳細を表示
4. **クリーンアップ** — ワンクリックで失敗したキーを削除して整理
